---
description: Automatically sync changes to the repository
---

1. Check for modified files

   ```bash
   git status
   ```

2. Add all changes

   ```bash
   git add .
   ```

3. Commit changes with a descriptive message based on the work done

   ```bash
   git commit -m "Update project with latest changes"
   ```

4. Push changes to the remote repository
   ```bash
   git push
   ```
